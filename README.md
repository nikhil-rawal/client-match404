
# Match404 - Frontend Client

Modern, responsive frontend application for the Match404 matching platform. Built with Next.js, React, TypeScript and Tailwind CSS for a type-safe user experience.

**[Link to Match404 - Server](https://github.com/nikhil-rawal/server-match404)**

## Tech Stack

- **Framework:** Next.js 16 with App Router
- **Library:** React 19 with React Compiler
- **Language:** TypeScript 5
- **Styling:** Tailwind CSS 4 + DaisyUI 5.5
- **State Management:** Zustand 5.0
- **Linting:** ESLint 9 with Next.js config

## Features

- **Modern UI/UX:** Clean, responsive design with DaisyUI components
- **Type Safety:** Full TypeScript implementation for robust code
- **Optimized Performance:** React Compiler for automatic optimization
- **Global State:** Lightweight state management with Zustand
- **Route Groups:** Organized authentication and protected routes
- **Server Components:** Leveraging Next.js App Router capabilities
- **Hot Reloading:** Fast development with Next.js dev server

## Pages & Routes

### Public Routes
- `/` - Landing page
- `/login` - User login
- `/signup` - User registration

### Protected Routes (Authenticated)
- `/feed` - Browse and discover matches
- `/profile` - View and edit user profile
- `/connections` - Manage connections and requests
- `/requests` - View received connection requests
